## 📖 앱 설명 (Description)

### 버전 1 (간결한 버전, 4000자 제한)

```
오늘 하루는 어땠나요?

바이브네컷은 당신의 일기를 AI가 네컷 만화로 만들어주는 특별한 앱입니다. 
복잡한 과정 없이, 그냥 오늘 있었던 일을 써내려가면 됩니다. 
나머지는 우리가 알아서 예쁘게 그려드릴게요.

✨ 간단한 사용법
1. 오늘 하루를 일기로 작성하세요
2. 원하는 만화 스타일을 선택하세요
3. AI가 자동으로 네컷 만화를 만들어드립니다
4. 갤러리에 저장하고 언제든 다시 볼 수 있어요

🎨 5가지 만화 스타일
• 일본 애니메이션 스타일: 밝고 생동감 넘치는 색감
• 수채화 느낌: 부드럽고 흐르는 듯한 톤
• 연필 스케치: 따뜻한 손그림 느낌
• 따뜻한 파스텔 톤: 포근하고 편안한 색감
• 고흐 화풍: 역동적이고 표현력 있는 붓터치

💝 특별한 기능
• 하루 한 번, 소중한 순간을 기록하세요
• AI가 생성한 감정 한 줄 문장으로 하루를 마무리
• 내가 만든 만화를 갤러리에서 모아보기
• 사진 앨범에 저장해서 친구들과 공유하기

📱 왜 바이브네컷인가요?
일기를 쓰는 건 좋은데, 매일 똑같은 텍스트만 쌓이는 게 아쉬셨나요? 
바이브네컷은 당신의 하루를 시각적으로 기록할 수 있게 해줍니다. 
작은 일상의 순간들이 네컷 만화로 변하면서, 
평범했던 하루도 특별한 추억이 됩니다.

🎯 이런 분들께 추천해요
• 일기를 쓰지만 더 특별하게 남기고 싶은 분
• SNS에 올릴 귀여운 콘텐츠를 찾는 분
• 일상을 시각적으로 기록하고 싶은 분
• AI 기술을 활용한 새로운 경험을 원하는 분

바이브네컷과 함께 오늘 하루도 소중한 추억으로 남겨보세요.
```

### 버전 2 (감성 중심 버전)

```
하루하루가 지나가면서, 
우리는 많은 순간들을 놓치고 있습니다.

바이브네컷은 그런 순간들을 
네컷 만화로 간직할 수 있게 해주는 앱입니다.

📝 일기만 쓰면 끝
복잡한 설정이나 어려운 기능은 없어요. 
그냥 오늘 있었던 일을 솔직하게 써내려가세요. 
AI가 당신의 이야기를 읽고, 
그 순간의 감정과 분위기를 네컷 만화로 그려드립니다.

🎨 나만의 스타일로
5가지 다른 만화 스타일 중에서 
오늘의 기분에 맞는 스타일을 선택할 수 있어요. 
밝고 경쾌한 하루라면 애니메이션 스타일, 
포근하고 편안한 하루라면 파스텔 톤으로.

💭 감정 한 줄 문장
만화가 완성되면, AI가 당신의 하루를 
따뜻하게 감싸주는 한 줄 문장을 만들어드립니다. 
"오늘 하루도 수고했어요" 같은 
작은 위로가 필요할 때가 있잖아요.

📸 영원히 간직하기
만든 만화는 갤러리에 자동으로 저장되고, 
사진 앨범에도 저장할 수 있어요. 
시간이 지나 다시 보면, 
그때의 감정이 생생하게 떠오를 거예요.

🌟 하루 한 번의 특별함
하루에 한 번만 일기를 쓸 수 있어요. 
그래서 더 소중하고, 더 특별하게 느껴집니다. 
오늘 하루를 정리하는 작은 의식처럼.

바이브네컷과 함께, 
당신의 일상이 만화가 되는 순간을 경험해보세요.
```
